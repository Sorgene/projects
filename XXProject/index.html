訪客推薦

綠朵休閒農場 | 今天小剎來介紹台中龍井夜景餐廳「綠朵休閒農場」去年開幕時已經有很多部落客衝去寫過了，真的蠻適合親子全家去的，孩子們可以有比較大的活動空間，大人也可以聊天放鬆，挺適合約會、溜小孩，大家快一起來看看吧~

小剎

User

台中夜景餐廳 綠朵餐廳最大賣點就是火紅的台中夕陽與絢爛的台中夜景，當天等到七點終於等到夕陽西下沈入海。 建議大家夏天可以六點到六點半之間進來 (假日要先定位)，這樣剛好可以看到美麗的夕陽晚霞，還可以看到臺中港火力發電廠的夜景。 整體來說，綠朵的景觀與用餐環境都滿舒適的，餐點口味以當天吃的來說就比較一般，這裡賣的重點是 View 景觀無價囉～

Angela

User

綠朵休閒農場位於大肚台地山坡上，附近夜景餐廳林立，而綠朵可說是規模最大的一家，走比較精緻的路線，不少新人也會選擇綠朵當作戶外婚禮的場地哦！ 綠朵最顯眼的除了白色建築，廣闊的綠草地也是一大賣點，不管是小孩或毛小孩，都很喜歡在這跑跑跳跳~ 綠朵有著那大片綠油油草原以及水池打亮的燈光~配上美美夜景~還可以在頂樓遠眺美麗的海線夜景待在這邊心情相當放鬆哦！
